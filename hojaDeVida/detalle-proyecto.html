Detalle del Proyecto Laberinto

Este aplicativo permite que el usuario navegue por el laberinto representado por los caracteres ASCII, teniendo en cuenta el uso de las flechas para desplazarse y así, encontrar la salida..

Características principales:

  • Creación, edición y eliminación de bloques.
  • Asignación de niveles con mayor dificultad (Próximamente).
  • Interfaz intuitiva y fácil de usar.

Tecnologías utilizadas:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python, SQL
  • Editor de Código: Visual Studio Code
  • Control de versiones: Git y GitHub

Desafíos y Soluciones:

Diseño de videojuego laberinto, implementando Python, SQL, Git y VSC, entre los desafios hacer que el laberinto corriera fluidamente, falta aún terminar de mejorarlo.
